**日期**：2026-03-13
**状态**：已确认

## 概述

一款运行在浏览器中的公路车 fitting 指导工具，使用笔记本摄像头实时捕捉骑行姿态，通过姿势检测分析三项核心 fitting 指标，以视觉叠加和语音提示的方式实时引导用户调整坐姿。面向个人骑手居家使用，无需安装，无需后端服务。

---

## 目标用户

- 个人公路车骑手
- 无专业 fitting 知识，希望自助完成基本 fitting 校准
- 使用场景：室内骑行台（trainer）训练时

## 页面与用户流程

### 1. 设置页

- 请求摄像头权限
- 提示最佳拍摄位置：侧面，距离约 2-3 米，摄像头与骑手腰部等高
- 选择可用摄像头设备（如有多个）

### 2. 校准页

- 实时显示摄像头画面
- MediaPipe Pose 持续检测人体关键点
- 检测到完整骨架后显示绿色确认指示
- 检测到稳定骑行姿态（骨架持续稳定 2 秒）后自动进入主界面
- 全程无需用户手动操作

### 3. 主骑行界面

**布局**：左右分栏

- **左侧（2/3 宽）**：摄像头实时画面 + Canvas 骨架叠加
  - 关节连线颜色：绿色（正常）/ 黄色（边缘）/ 红色（偏离）
  - 三处角度数值以浮动标签显示在对应关节旁
- **右侧（1/3 宽）**：三项指标卡片，竖向排列
  - 指标名称 + 当前角度数值（大字）
  - 进度条显示当前值在目标范围内的位置
  - 目标范围文字说明

**退出**：按空格键或 ESC 停止录制，进入总结页

### 4. Session 总结页

- 本次骑行的三项指标平均值 vs 目标范围
- 3-5 条文字调整建议（基于偏离方向生成，如"坐垫偏低，建议上调 5-10mm"）
- 保存本次 session 数据到 localStorage（时间戳 + 平均角度）
- 可查看历史 session 列表（简单列表，显示日期和综合评分）

## 指标定义

### 膝盖角度（Knee Extension Angle）

- **计算**：髋关节 → 膝关节 → 踝关节 三点夹角
- **测量时机**：踩踏最低点（死点）附近的平均值（曲柄 6 点钟位置 ±15°）
- **目标范围**：140° – 150°
- **偏低（< 140°）**：坐垫偏低，建议升高坐垫
- **偏高（> 150°）**：坐垫偏高或后移，建议降低坐垫

### 躯干角度（Torso Angle）

- **计算**：肩关节 → 髋关节 连线与垂直方向的夹角
- **测量时机**：持续采样，取滑动平均值
- **目标范围**：35° – 45°（相对垂直）
- **偏小（< 35°）**：上体过于前倾，可能需要调高把立或缩短车把距
- **偏大（> 45°）**：上体过于直立，可能需要降低把立或增加前伸量

### 手肘角度（Elbow Angle）

- **计算**：肩关节 → 肘关节 → 腕关节 三点夹角
- **测量时机**：持续采样，取滑动平均值
- **目标范围**：150° – 165°
- **偏小（< 150°）**：手肘弯曲过多，可能车把距偏短
- **偏大（> 165°）**：手肘接近伸直，可能车把距偏长或把立偏高

## 反馈系统

### 视觉反馈

- Canvas 实时重绘，帧率跟随 MediaPipe 检测帧率（目标 30fps）
- 骨架关键线段按指标状态着色：
  - 绿色：在目标范围内
  - 黄色：偏离 5° 以内（边缘警告）
  - 红色：偏离超过 5°
- 角度数值标签实时更新，显示在对应关节附近

### 语音反馈

- 使用 Web Speech API（中文语音）
- 触发条件：某项指标偏离目标范围超过 **3 秒**
- 去重：同一条提示 **30 秒** 内不重复播报
- 优先级：膝盖角度 > 躯干角度 > 手肘角度（同时偏离时先播报最重要的）
- 示例播报内容：
  - "膝盖角度偏小，建议升高坐垫"
  - "上体过于直立，注意前倾姿势"
  - "手肘弯曲过多，尝试放松手臂"

## 数据模型（localStorage）

```typescript
interface Session {
  id: string;           // UUID
  timestamp: number;    // Unix timestamp
  duration: number;     // 骑行秒数
  avgKneeAngle: number;
  avgTorsoAngle: number;
  avgElbowAngle: number;
  score: number;        // 综合评分 0-100，基于三项指标的偏离程度
}

// localStorage key: "bike-fitting-sessions"
// 存储格式: Session[]，最多保存 20 条
```

## 成功标准

1. 在 Chrome/Safari 中无需安装即可运行
2. MediaPipe 检测延迟 < 100ms（MacBook 2019+ 环境）
3. 三项角度计算误差 < 5°（与手动量角器对比）
4. 语音提示不干扰骑行体验（触发阈值和去重逻辑合理）
5. 初次使用用户（无引导）能在 3 分钟内开始第一次骑行分析
